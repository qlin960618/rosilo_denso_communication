# rosilo_denso_communication
Communicate with DENSO robots over ROS.

## Requirements

### Rosilo
- `rosilo_datalogger`
- `rosilo_clock`
- `rosilo_conversions`
- `rosilo_robot_driver_denso`

### DQ Robotics
See installation instructions at: https://dqrobotics.github.io/
- `libdqrobotics`
- `libdqrobotics-interface-vrep`
